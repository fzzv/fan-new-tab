# Fan-new-tab 新标签页

## 介绍
这是我写的第一个浏览器插件，用于练习如何实现浏览器插件

它是一个新标签页插件，可以在新标签页中进行搜索和显示收藏的网页书签

## 技术栈
- 🥝 [Vue3](https://vuejs.org/)
- 🦾 [TypeScript](https://www.typescriptlang.org/)
- 🧱 [Vite](https://vite.dev/)

## 功能
- [ ] 搜索
- [ ] 书签

## 本地调试
```bash
npm i
npm run dev
```
将 `extension` 目录拖动到浏览器的扩展程序中即可
